# Next.js Client Data App

A Next.js 14 application demonstrating server-side and client-side data fetching patterns using the App Router.

## Overview
Built with Next.js App Router, this project explores rendering strategies including Server Components and Client Components, with a dedicated route for client-side data fetching.

## Features
- Next.js 14 App Router with nested layouts
- Client-side data fetching with React hooks
- TypeScript throughout
- Tailwind CSS styling
- ESLint configuration

## Technologies Used
- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Linting:** ESLint

## Project Structure
```
src/app/
+-- layout.tsx          # Root layout with global styles
+-- page.tsx            # Home page (Server Component)
+-- globals.css         # Global Tailwind styles
+-- client-data/
    +-- page.tsx        # Client-side data fetching demo
```

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
